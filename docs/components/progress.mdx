# Progress

Die Casoon UI Library bietet flexible Progress-Komponenten für Fortschrittsanzeigen.

## Standard Progress

```html
<!-- Linear Progress -->
<div class="progress">
  <div class="progress-bar" style="width: 50%">
    <span class="progress-label">50%</span>
  </div>
</div>

<!-- Circular Progress -->
<div class="progress progress--circular">
  <div class="progress-circle" style="--progress: 50%">
    <span class="progress-label">50%</span>
  </div>
</div>
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie ARIA-Attribute
   - Stellen Sie sicher, dass der Fortschritt auch textuell verfügbar ist
   - Implementieren Sie Screenreader-Unterstützung

2. **Responsive**:
   - Optimieren Sie die Darstellung für mobile Geräte
   - Verwenden Sie angemessene Größen
   - Stellen Sie sicher, dass der Fortschritt gut sichtbar ist

3. **Performance**:
   - Vermeiden Sie komplexe Animationen
   - Optimieren Sie die Rendering-Performance
   - Aktualisieren Sie den Fortschritt effizient

4. **UX**:
   - Bieten Sie klare visuelle Hinweise
   - Zeigen Sie den Fortschritt in Echtzeit
   - Geben Sie Feedback bei Abschluss
