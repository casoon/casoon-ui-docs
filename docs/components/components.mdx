## Installation

```bash
npm install casoon-ui-lib
```

## Verwendung

```css
/* Basis-Styles */
@import 'casoon-ui-lib/styles/base.css';

/* Module */
@import 'casoon-ui-lib/modules/button.module.css';
@import 'casoon-ui-lib/modules/input.module.css';
@import 'casoon-ui-lib/modules/modal.module.css';

/* Theme */
@import 'casoon-ui-lib/themes/default.css';
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie ARIA-Attribute
   - Stellen Sie sicher, dass alle Komponenten mit Screenreadern kompatibel sind
   - Implementieren Sie Tastatur-Navigation

2. **Responsive**:
   - Optimieren Sie die Komponenten für mobile Geräte
   - Verwenden Sie flexible Einheiten
   - Testen Sie auf verschiedenen Bildschirmgrößen

3. **Performance**:
   - Laden Sie nur benötigte Module
   - Optimieren Sie Animationen
   - Vermeiden Sie unnötige DOM-Manipulationen

4. **UX**:
   - Halten Sie die Benutzeroberfläche konsistent
   - Bieten Sie klare Feedback-Mechanismen
   - Implementieren Sie sinnvolle Standardwerte
