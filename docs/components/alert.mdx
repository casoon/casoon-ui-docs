# Alert

Die Casoon UI Library bietet flexible Alert-Komponenten für Benachrichtigungen und Feedback.

## Installation

```bash
npm install casoon-ui-lib
```

## Verwendung

```css
@import 'casoon-ui-lib/modules/alert.module.css';
```

## Beispiele

### Standard Alert

```html
<!-- Info Alert -->
<div class="alert">
  <div class="alert__icon">
    <i class="icon icon--info"></i>
  </div>
  <div class="alert__content">
    <h4 class="alert__title">Information</h4>
    <p class="alert__message">Dies ist eine informative Nachricht.</p>
  </div>
  <button class="alert__close">
    <i class="icon icon--close"></i>
  </button>
</div>
```

### Aktionen

```html
<!-- Alert mit Button -->
<div class="alert">
  <div class="alert__content">
    <p class="alert__message">Möchten Sie diese Aktion ausführen?</p>
  </div>
  <div class="alert__actions">
    <button class="button button--primary">Bestätigen</button>
    <button class="button button--secondary">Abbrechen</button>
  </div>
</div>

<!-- Alert mit Link -->
<div class="alert">
  <div class="alert__content">
    <p class="alert__message">Neue Version verfügbar!</p>
    <a href="#" class="alert__link">Jetzt aktualisieren</a>
  </div>
</div>
```

### Animationen

```html
  <div class="alert__content">

  </div>

    <p class="alert__message">Diese Nachricht wird eingeschoben.</p>
  </div>
</div>
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie ARIA-Attribute für Status und Rolle
   - Stellen Sie sicher, dass der Alert-Text für Screenreader verfügbar ist
   - Implementieren Sie Tastatur-Navigation

2. **Responsive**:
   - Optimieren Sie das Layout für mobile Geräte
   - Verwenden Sie angemessene Abstände
   - Passen Sie die Schriftgrößen an

3. **Performance**:
   - Vermeiden Sie zu viele animierte Alerts
   - Optimieren Sie die Animationen
   - Entfernen Sie nicht mehr benötigte Alerts

4. **UX**:
   - Verwenden Sie klare und prägnante Nachrichten
   - Bieten Sie Handlungsmöglichkeiten an
   - Geben Sie Feedback bei Interaktionen
