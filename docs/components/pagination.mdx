# Pagination

Die Casoon UI Library bietet flexible Paginierungskomponenten für die Navigation durch große Datensätze.

## Standard Pagination

```html
<nav class="pagination">
  <ul class="pagination-list">
    <li>
      <a href="#" class="pagination-item pagination-prev" disabled>
        <span class="pagination-icon">←</span>
        Vorherige
      </a>
    </li>
    <li>
      <a href="#" class="pagination-item is-active">1</a>
    </li>
    <li>
      <a href="#" class="pagination-item">2</a>
    </li>
    <li>
      <a href="#" class="pagination-item">3</a>
    </li>
    <li>
      <span class="pagination-ellipsis">...</span>
    </li>
    <li>
      <a href="#" class="pagination-item">10</a>
    </li>
    <li>
      <a href="#" class="pagination-item pagination-next">
        Nächste
        <span class="pagination-icon">→</span>
      </a>
    </li>
  </ul>
</nav>
```

## Pagination Varianten

```html
<!-- Rounded Pagination -->
<nav class="pagination pagination--rounded">
  <!-- Pagination-Inhalt -->
</nav>

<!-- Compact Pagination -->
<nav class="pagination pagination--compact">
  <!-- Pagination-Inhalt -->
</nav>

<!-- Simple Pagination -->
<nav class="pagination pagination--simple">
  <a href="#" class="pagination-item pagination-prev" disabled>←</a>
  <a href="#" class="pagination-item pagination-next">→</a>
</nav>
```

## Pagination mit Dropdown

```html
<nav class="pagination">
  <div class="pagination-info">
    Zeige
    <select class="select select--sm">
      <option>10</option>
      <option>25</option>
      <option>50</option>
      <option>100</option>
    </select>
    Einträge
  </div>
  <ul class="pagination-list">
    <!-- Pagination-Inhalt -->
  </ul>
</nav>
```

## Pagination mit Info

```html
<nav class="pagination">
  <div class="pagination-info">
    Zeige 1 bis 10 von 100 Einträgen
  </div>
  <ul class="pagination-list">
    <!-- Pagination-Inhalt -->
  </ul>
</nav>
```


## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie semantische HTML-Elemente
   - Fügen Sie ARIA-Labels hinzu
   - Stellen Sie sicher, dass die Navigation mit der Tastatur möglich ist

2. **Responsive**:
   - Optimieren Sie die Darstellung für mobile Geräte
   - Verwenden Sie angemessene Abstände
   - Stellen Sie sicher, dass die Elemente gut klickbar sind

3. **Performance**:
   - Lazy-Loaden Sie die Paginierungsdaten
   - Vermeiden Sie unnötige Neuberechnungen
   - Optimieren Sie die Rendering-Performance

4. **UX**:
   - Zeigen Sie den aktuellen Status an
   - Bieten Sie klare Navigationselemente
   - Geben Sie Feedback bei Benutzerinteraktionen
