## Installation


## Verwendung

```astro
---

---

<div class="sidebar">
  <div class="sidebar-header">
    <h3 class="sidebar-title">Navigation</h3>
  </div>

    <a href="#" class="sidebar-link active">Dashboard</a>

```

## Varianten

### Standard Sidebar

```html
<div class="sidebar">
  <nav class="sidebar-nav">
    <a href="#" class="sidebar-link">Link 1</a>
    <a href="#" class="sidebar-link">Link 2</a>
    <a href="#" class="sidebar-link">Link 3</a>
  </nav>
</div>
```

### Sidebar mit Header und Footer

```html
<div class="sidebar">
  <div class="sidebar-header">
    <h3 class="sidebar-title">Titel</h3>
  </div>
  <nav class="sidebar-nav">
    <a href="#" class="sidebar-link">Link 1</a>
    <a href="#" class="sidebar-link">Link 2</a>
  </nav>
  <div class="sidebar-footer">
    <button class="btn btn-primary">Aktion</button>
  </div>
</div>
```

### Collapsible Sidebar

```html
<div class="sidebar sidebar-collapsible">
  <button class="sidebar-toggle">
    <span class="sidebar-toggle-icon"></span>
  </button>
  <nav class="sidebar-nav">
    <a href="#" class="sidebar-link">Link 1</a>
    <a href="#" class="sidebar-link">Link 2</a>
  </nav>
</div>
```

## CSS-Variablen

```css
:root {
  --sidebar-width: 250px;
  --sidebar-collapsed-width: 60px;
  --sidebar-background: var(--color-white);
  --sidebar-border: 1px solid var(--color-gray-200);
  --sidebar-shadow: var(--shadow-sm);
}
```

## Anpassung

```css
:root {
  --sidebar-width: 300px;
  --sidebar-background: var(--color-gray-50);
  --sidebar-border: none;
}
```
