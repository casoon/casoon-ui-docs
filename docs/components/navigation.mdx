# Navigation

Die Casoon UI Library bietet verschiedene Navigationskomponenten für Ihre Anwendung.

## Navbar

```html
<!-- Standard Navbar -->
<nav class="navbar">
  <div class="navbar-brand">
    <a href="/" class="navbar-logo">Logo</a>
  </div>
  <div class="navbar-menu">
    <a href="#" class="navbar-item">Home</a>
    <a href="#" class="navbar-item">Über uns</a>
    <a href="#" class="navbar-item">Kontakt</a>
  </div>
  <div class="navbar-end">
    <button class="btn">Login</button>
  </div>
</nav>

<!-- Responsive Navbar -->
<nav class="navbar">
  <div class="navbar-brand">
    <a href="/" class="navbar-logo">Logo</a>
    <button class="navbar-burger">
      <span></span>
      <span></span>
      <span></span>
    </button>
  </div>
  <div class="navbar-menu is-active">
    <!-- Menü-Inhalt -->
  </div>
</nav>
```

## Breadcrumb

```html
<!-- Standard Breadcrumb -->
<nav class="breadcrumb">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Produkte</a></li>
    <li><span>Details</span></li>
  </ul>
</nav>

<!-- Breadcrumb mit Icons -->
<nav class="breadcrumb">
  <ul>
    <li>
      <a href="#">
        <span class="breadcrumb-icon">🏠</span>
        Home
      </a>
    </li>

```

## Tabs

```html
<!-- Standard Tabs -->
<div class="tabs">
  <ul class="tabs-nav">
    <li class="is-active"><a href="#tab1">Tab 1</a></li>
    <li><a href="#tab2">Tab 2</a></li>
    <li><a href="#tab3">Tab 3</a></li>
  </ul>
  <div class="tabs-content">
    <div id="tab1" class="tabs-pane is-active">
      <p>Inhalt Tab 1</p>
    </div>
    <div id="tab2" class="tabs-pane">
      <p>Inhalt Tab 2</p>
    </div>
    <div id="tab3" class="tabs-pane">
      <p>Inhalt Tab 3</p>
    </div>
  </div>
</div>
```

## CSS Variablen

```css
:root {
  /* Navbar */
  --navbar-height: 4rem;
  --navbar-padding: var(--spacing-4);
  --navbar-bg: var(--white);
  --navbar-shadow: var(--shadow-sm);
  
  /* Sidebar */
  --sidebar-width: 16rem;
  --sidebar-collapsed-width: 4rem;
  --sidebar-bg: var(--gray-50);
  --sidebar-item-padding: var(--spacing-3);
  
  /* Breadcrumb */
  --breadcrumb-padding: var(--spacing-2);
  --breadcrumb-separator: "/";
  --breadcrumb-color: var(--gray-600);
  
  /* Tabs */
  --tabs-border-color: var(--gray-200);
  --tabs-active-color: var(--primary);
  --tabs-padding: var(--spacing-4);
}
```

## Best Practices

1. **Zugänglichkeit**: Verwenden Sie semantische HTML-Elemente
2. **Responsive**: Stellen Sie sicher, dass die Navigation auf allen Geräten gut bedienbar ist
3. **Konsistenz**: Halten Sie sich an die vordefinierten Navigationsmuster
4. **Performance**: Optimieren Sie die Navigation für schnelle Ladezeiten
