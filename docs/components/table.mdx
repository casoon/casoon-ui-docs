# Table

Die Casoon UI Library bietet flexible Tabellen-Komponenten für die Datenanzeige.

## Standard Table

```html
<table class="table">
  <thead>
    <tr>
      <th>Name</th>
      <th>E-Mail</th>
      <th>Rolle</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Max Mustermann</td>
      <td>max@example.com</td>
      <td>Admin</td>
    </tr>
    <tr>
      <td>Erika Musterfrau</td>
      <td>erika@example.com</td>
      <td>User</td>
    </tr>
  </tbody>
</table>
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie semantische HTML-Elemente
   - Fügen Sie Tabellenbeschreibungen hinzu
   - Stellen Sie sicher, dass die Tabelle mit Screenreadern kompatibel ist

2. **Responsive**:
   - Verwenden Sie die responsive Wrapper-Komponente
   - Optimieren Sie die Spaltenbreiten
   - Stellen Sie sicher, dass der Inhalt auf mobilen Geräten lesbar ist

3. **Performance**:
   - Begrenzen Sie die Anzahl der Zeilen
   - Implementieren Sie Paginierung für große Datensätze
   - Optimieren Sie die Rendering-Performance

4. **UX**:
   - Bieten Sie Sortier- und Filteroptionen
   - Zeigen Sie klare Aktionen an
   - Geben Sie Feedback bei Benutzerinteraktionen
