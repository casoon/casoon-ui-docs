# Effects

Die Casoon UI Library bietet verschiedene Effekte für Animationen und Übergänge.

## Transitions

```html
<!-- Fade -->
<div class="transition-fade">Fade Effekt</div>

<!-- Slide -->
<div class="transition-slide">Slide Effekt</div>

<!-- Scale -->
<div class="transition-scale">Scale Effekt</div>

<!-- Rotate -->
<div class="transition-rotate">Rotate Effekt</div>
```

## Animations

```html
<!-- Pulse -->
<div class="animate-pulse">Pulsierender Effekt</div>

<!-- Bounce -->
<div class="animate-bounce">Springender Effekt</div>

<!-- Spin -->
<div class="animate-spin">Drehender Effekt</div>

<!-- Ping -->
<div class="animate-ping">Ping Effekt</div>
```

## Hover Effects

```html
<!-- Hover Scale -->
<div class="hover-scale">Hover Scale</div>

<!-- Hover Rotate -->
<div class="hover-rotate">Hover Rotate</div>

<!-- Hover Shadow -->
<div class="hover-shadow">Hover Shadow</div>

<!-- Hover Opacity -->
<div class="hover-opacity">Hover Opacity</div>
```

## Focus Effects

```html
<!-- Focus Ring -->
<input class="focus-ring" type="text">

<!-- Focus Scale -->
<input class="focus-scale" type="text">

<!-- Focus Shadow -->
<input class="focus-shadow" type="text">
```

## CSS Variablen

```css
:root {
  /* Transition */
  --transition-duration: 150ms;
  --transition-timing: cubic-bezier(0.4, 0, 0.2, 1);
  
  /* Animation */
  --animation-duration: 1s;
  --animation-timing: ease-in-out;
  
  /* Hover */
  --hover-scale: 1.05;
  --hover-rotate: 5deg;
  --hover-shadow: var(--shadow-lg);
  --hover-opacity: 0.8;
  
  /* Focus */
  --focus-ring-width: 2px;
  --focus-ring-color: var(--primary);
  --focus-ring-offset: 2px;
}
```

## Best Practices

1. **Performance**: Nutzen Sie Hardware-beschleunigte Animationen
2. **Zugänglichkeit**: Stellen Sie sicher, dass Animationen deaktivierbar sind
3. **Konsistenz**: Halten Sie sich an die vordefinierten Effekte
4. **Responsive**: Testen Sie Effekte auf verschiedenen Geräten
