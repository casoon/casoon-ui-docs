## Formular-Gruppen

```html
<div class="form-group">
  <label for="input" class="form-label">Label</label>
  <input type="text" id="input" class="form-control" placeholder="Platzhalter">
  <small class="form-text">Hilfetext</small>
</div>
```

## Input-Varianten

### Standard Input

```html
<input type="text" class="form-control" placeholder="Standard Input">
```

### Disabled Input

```html
<input type="text" class="form-control" placeholder="Disabled Input" disabled>
```

### Readonly Input

```html
<input type="text" class="form-control" value="Readonly Input" readonly>
```

## Checkbox und Radio

```html
<div class="form-check">
  <input type="checkbox" id="checkbox" class="form-check-input">
  <label for="checkbox" class="form-check-label">Checkbox</label>
</div>

<div class="form-check">
  <input type="radio" id="radio" name="radio" class="form-check-input">
  <label for="radio" class="form-check-label">Radio Button</label>
</div>
```

## Select

```html
<select class="form-select">
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
```

## CSS-Variablen

```css
:root {
  --form-control-padding: var(--spacing-2) var(--spacing-3);
  --form-control-border-radius: var(--radius-md);
  --form-control-border: 1px solid var(--color-gray-300);
  --form-control-background: var(--color-white);
  --form-control-color: var(--color-text);
}
```

## Anpassung

```css
:root {
  --form-control-padding: 1rem;
  --form-control-border-radius: 0.5rem;
  --form-control-background: var(--color-gray-50);
}
```
