# Toast

Die Casoon UI Library bietet flexible Toast-Komponenten für kurze Benachrichtigungen.

## Installation

```bash
npm install casoon-ui-lib
```

## Verwendung

```css
@import 'casoon-ui-lib/modules/toast.module.css';
```

## Beispiele

### Standard Toast

```html
<!-- Info Toast -->
<div class="toast">
  <div class="toast__icon">
    <i class="icon icon--info"></i>
  </div>
  <div class="toast__content">
    <p class="toast__message">Dies ist eine kurze Nachricht.</p>
  </div>
  <button class="toast__close">
    <i class="icon icon--close"></i>
  </button>
</div>
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie ARIA-Attribute für Status und Rolle
   - Stellen Sie sicher, dass der Toast-Text für Screenreader verfügbar ist
   - Implementieren Sie Tastatur-Navigation

2. **Responsive**:
   - Optimieren Sie die Größe für mobile Geräte
   - Verwenden Sie angemessene Abstände
   - Passen Sie die Positionierung bei Bedarf an

3. **Performance**:
   - Begrenzen Sie die Anzahl gleichzeitiger Toasts
   - Optimieren Sie die Animationen
   - Entfernen Sie nicht mehr benötigte Toasts

4. **UX**:
   - Halten Sie die Nachrichten kurz und prägnant
   - Verwenden Sie Toasts sparsam
   - Geben Sie Feedback bei Interaktionen
