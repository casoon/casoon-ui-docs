# Slider

Die Slider-Komponente bietet eine interaktive Möglichkeit, Werte auszuwählen oder Inhalte zu präsentieren.

## Installation

```bash
npm install casoon-ui-lib
```

## Verwendung

```astro
---
import 'casoon-ui-lib/modules/slider.module.css';
---

<div class="slider">
  <div class="slider-track">
    <div class="slider-fill" style="width: 50%"></div>
    <div class="slider-thumb" role="slider" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
  </div>
</div>
```

## Varianten

### Standard Slider

```html
<div class="slider">
  <div class="slider-track">
    <div class="slider-fill" style="width: 30%"></div>
    <div class="slider-thumb" role="slider" aria-valuenow="30"></div>
  </div>
</div>
```

### Range Slider

```html
<div class="slider slider-range">
  <div class="slider-track">
    <div class="slider-fill" style="left: 20%; width: 40%"></div>
    <div class="slider-thumb" role="slider" aria-valuenow="20"></div>
    <div class="slider-thumb" role="slider" aria-valuenow="60"></div>
  </div>
</div>
```

### Disabled Slider

```html
<div class="slider" disabled>
  <div class="slider-track">
    <div class="slider-fill" style="width: 50%"></div>
    <div class="slider-thumb" role="slider" aria-valuenow="50" aria-disabled="true"></div>
  </div>
</div>
```

## CSS-Variablen

```css
:root {
  --slider-height: 4px;
  --slider-thumb-size: 16px;
  --slider-track-background: var(--color-gray-200);
  --slider-fill-background: var(--color-primary);
  --slider-thumb-background: var(--color-white);
  --slider-thumb-border: 2px solid var(--color-primary);
}
```

## Anpassung

```css
:root {
  --slider-height: 6px;
  --slider-thumb-size: 20px;
  --slider-track-background: var(--color-gray-300);
  --slider-fill-background: var(--color-secondary);
}
```

## Zugänglichkeit

- Slider ist mit Tastatur navigierbar
- ARIA-Attribute für Screen Reader
- Visuelles Feedback für Interaktionen
- Fokus-States sind deutlich erkennbar
