# Utilities

Die Casoon UI Library bietet eine Vielzahl von Utility-Klassen für schnelles Styling.

## Display

```html
<!-- Block -->
<div class="block">Block Element</div>

<!-- Inline -->
<span class="inline">Inline Element</span>

<!-- Flex -->
<div class="flex">Flex Container</div>

<!-- Grid -->
<div class="grid">Grid Container</div>

<!-- Hidden -->
<div class="hidden">Verstecktes Element</div>
```

<!-- Fixed -->

<div class="sticky">Klebrig</div>

## CSS Variablen

```css
:root {
  /* Z-Index */
  --z-0: 0;
  --z-10: 10;
  --z-20: 20;
  --z-50: 50;
  
  /* Shadow */
  --shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
  --shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
  
  /* Border Radius */
  --radius-sm: 0.125rem;
  --radius: 0.25rem;
  --radius-lg: 0.5rem;
  --radius-full: 9999px;
}
```

## Best Practices

1. **Konsistenz**: Nutzen Sie die vordefinierten Utility-Klassen
2. **Performance**: Vermeiden Sie zu viele verschachtelte Utility-Klassen
3. **Wartbarkeit**: Kombinieren Sie Utility-Klassen sinnvoll
4. **Responsive**: Nutzen Sie responsive Varianten der Utilities

## Sichtbarkeit

```css
.visible { visibility: visible; }
.invisible { visibility: hidden; }
```

## Pointer Events

```css
.pointer-events-none { pointer-events: none; }
.pointer-events-auto { pointer-events: auto; }
```

## User Select

```css

.select-all { user-select: all; }
```

## Verwendung

```html

  <div class="relative">
    <div class="absolute top-0 right-0 z-10">
      <span class="cursor-pointer select-none">×</span>
    </div>
  </div>
</div>

<div class="overflow-hidden">
  <div class="opacity-75">
    Halbtransparenter Inhalt
  </div>
</div>
```

## Responsive Utilities

```css
@media (min-width: 640px) {
  .sm\:hidden { display: none; }
  .sm\:block { display: block; }
}

@media (min-width: 768px) {
  .md\:hidden { display: none; }
  .md\:block { display: block; }
}

@media (min-width: 1024px) {
  .lg\:hidden { display: none; }
  .lg\:block { display: block; }
}
```
