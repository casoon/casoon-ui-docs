# Typografie

Die Typografie-System der Casoon UI Library bietet eine konsistente und zugängliche Textgestaltung.

## Schriftfamilien

```css
:root {
  --font-family-sans: system-ui, -apple-system, sans-serif;
  --font-family-mono: ui-monospace, monospace;
}
```

## Verwendung

```html
<h1>Überschrift Level 1</h1>
<p class="text-lg text-muted">Ein längerer Textabsatz mit größerer Schrift.</p>
<div class="text-center">
  <h2>Zentrierte Überschrift</h2>
  <p>Zentrierter Text darunter</p>
</div>
```

## Responsive Typografie

```css
@media (min-width: 640px) {
  h1 {
    font-size: var(--font-size-5xl);
  }
  
  .text-base {
    font-size: var(--font-size-lg);
  }
}
```

## Best Practices

1. **Hierarchie**: Verwenden Sie die Überschriften-Hierarchie korrekt
2. **Lesbarkeit**: Achten Sie auf ausreichende Zeilenhöhe und Kontrast
3. **Konsistenz**: Halten Sie sich an die vordefinierten Textstile
4. **Zugänglichkeit**: Stellen Sie sicher, dass die Typografie für Screenreader optimiert ist

## CSS Variablen

```css
:root {
  /* Schriftfamilien */
  --font-family-base: system-ui, -apple-system, sans-serif;
  --font-family-heading: var(--font-family-base);
  
  /* Schriftgrößen */
  --font-size-xs: 0.75rem;
  --font-size-sm: 0.875rem;
  --font-size-base: 1rem;
  --font-size-lg: 1.125rem;
  --font-size-xl: 1.25rem;
  
  /* Zeilenhöhen */
  --line-height-tight: 1.25;
  --line-height-normal: 1.5;
  --line-height-relaxed: 1.75;
}
```
