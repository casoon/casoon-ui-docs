# Tooltip

Die Casoon UI Library bietet flexible Tooltip-Komponenten für zusätzliche Informationen.

## Standard Tooltip

```html
<!-- Text Tooltip -->
<button class="button" data-tooltip="Dies ist ein Tooltip">
  Hover mich
</button>

<!-- HTML Tooltip -->
<button class="button" data-tooltip-html="<strong>Fett</strong> und <em>kursiv</em>">
  HTML Tooltip
</button>
```

## Tooltip Position

```html
<!-- Top Tooltip -->
<button class="button" data-tooltip="Oben" data-tooltip-position="top">
  Top
</button>

<!-- Right Tooltip -->
<button class="button" data-tooltip="Rechts" data-tooltip-position="right">

</button>

<!-- Bottom Tooltip -->

  Unten
</button>

<!-- Left Tooltip -->
<button class="button" data-tooltip="Links" data-tooltip-position="left">
  Links
</button>
```

## Best Practices

1. **Zugänglichkeit**:
   - Verwenden Sie ARIA-Attribute
   - Stellen Sie sicher, dass der Tooltip-Text für Screenreader verfügbar ist
   - Implementieren Sie Tastatur-Navigation

2. **Responsive**:
   - Optimieren Sie die Position für mobile Geräte
   - Passen Sie die Größe an den verfügbaren Platz an
   - Vermeiden Sie Überlappungen

3. **Performance**:
   - Vermeiden Sie zu viele Tooltips gleichzeitig
   - Optimieren Sie die Animationen
   - Verwenden Sie CSS-Transitions statt JavaScript

4. **UX**:
   - Halten Sie den Text kurz und prägnant
   - Verwenden Sie Tooltips sparsam
   - Stellen Sie sicher, dass der Zweck klar ist
