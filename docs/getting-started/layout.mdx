# Layout

Das Layout-System der Casoon UI Library bietet flexible und responsive Layout-Komponenten.

## Container

```html
<div class="container">
  <div class="container__content">
    <!-- Inhalt -->
  </div>
</div>
```

### Container Varianten

```html
<!-- Standard Container -->
<div class="container">
  <div class="container__content">Standard</div>
</div>

<!-- Container mit max-width -->
<div class="container container--max-width">
  <div class="container__content">Max Width</div>
</div>

<!-- Container mit Padding -->
<div class="container container--padding">
  <div class="container__content">Mit Padding</div>
</div>
```

## Grid System

```html
<div class="grid">
  <div class="col col--4">Spalte 1</div>
  <div class="col col--4">Spalte 2</div>
  <div class="col col--4">Spalte 3</div>
</div>
```

### Responsive Grid

```html
<div class="grid grid--responsive">
  <div class="col col--sm-12 col--md-6 col--lg-4">
    Responsive Spalte
  </div>
</div>
```

## Flexbox

```html
<!-- Flex Container -->
<div class="flex">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

<!-- Flex mit Ausrichtung -->
<div class="flex flex--justify-between flex--align-center">
  <div>Links</div>
  <div>Mitte</div>
  <div>Rechts</div>
</div>
```

## Spacing

```html
<!-- Margin -->
<div class="m-4">Margin 4</div>
<div class="mt-2">Margin Top 2</div>
<div class="mx-auto">Auto Margin X</div>

<!-- Padding -->
<div class="p-4">Padding 4</div>
<div class="pt-2">Padding Top 2</div>
```

## CSS Variablen

```css
:root {
  /* Container */
  --container-max-width: 1200px;
  --container-padding: var(--spacing-4);
  
  /* Grid */
  --grid-columns: 12;
  --grid-gap: var(--spacing-4);
  
  /* Breakpoints */
  --breakpoint-sm: 640px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
}
```

## Best Practices

1. **Responsive Design**: Nutzen Sie die vordefinierten Breakpoints
2. **Konsistenz**: Halten Sie sich an die Grid-Struktur
3. **Performance**: Vermeiden Sie verschachtelte Grids
4. **Zugänglichkeit**: Stellen Sie sicher, dass das Layout für Screenreader optimiert ist
