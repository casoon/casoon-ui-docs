# Casoon UI Library

Die Casoon UI Library ist eine moderne, flexible und zugängliche Komponentenbibliothek.

## Installation

```bash
# Repository klonen
git init
git clone https://github.com/your-username/casoon-ui-lib.git

# Abhängigkeiten installieren
npm install casoon-ui-lib
```

## Verwendung

```css
/* Basis-Styles */
@import 'casoon-ui-lib/styles/base.css';

/* Module */
@import 'casoon-ui-lib/modules/button.module.css';
@import 'casoon-ui-lib/modules/input.module.css';
@import 'casoon-ui-lib/modules/modal.module.css';

/* Theme */
@import 'casoon-ui-lib/themes/default.css';
```

- Chrome (letzte 2 Versionen)
- Firefox (letzte 2 Versionen)
- Safari (letzte 2 Versionen)
